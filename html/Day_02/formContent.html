사용자의 정보를 입력받는 태그

form 요소는 사용자의 데이터를 입력받는 요소들을 의미한다.
form 요소의 대부분은 인라인 요소이다.
회원가입화면, 로그인화면, 회원정보수정화면, 검색창 등이 모두 폼관련 태그로 구성된다.

input태그 중 입력상자 : type="text", type="password", type="hidden"

사용자로부터 직접 입력

type="radio": 여러 항목 중에 하나만 선택할 때 사용

Location 서울 부산 제주

select / option 태그: 여러 항목 중에 하나만 선택할 때

출신학교와 선호색상을 선택하세요

input type="checkbox" : 여러 항목 중에 여러 개 선택할 때

취미를 선택하세요 낚시   자전거타기   수영   등산   캠핑  

select / option : multiple속성을 이용해서 여러 개 선택

textarea 태그 - 여러 줄 글상자



    input type의 속성: input tag의 성격을 정의한다.

  • type="text": 한 줄짜리 텍스트를 입력하기 위한 용도
  • type="password": 비밀번호 등 외부에 노출되지 않는 정보를 입력하기 위한 용도
  • type="hidden": 사용자에게 보이지는 않지만 서버로 넘어가는 정보를 정의하는 필드 (반드시 name과 value값을 지정해야함)
  • type="button": 버튼 모양을 생성. 아무런 기능 없음
    기능을 가지고 있는 경우 자바스크립트와 연동시 불편함이 있을 수 있어 자바스크립트와 편리한 연동을 위함
  • type="submit": form action 속성이 지정한 url 혹은 서버프로그램(경로)로 데이터를 전송
  • type="reset": 리셋 버튼 기능, 입력 초기화
  • type="file": 사용자가 파일을 첨부해서 서버로 전송하기 위한 버튼
    반드시 2가지 정보를 추가해야함
    1)form 태그의 method속성값은 반드시 post이어야함
    2)form 태그의 enctype속성값에 multipart/form-data값을 추가해야함
