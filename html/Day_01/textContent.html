텍스트 콘텐츠를 표기하는 태그

html문서의 body 태그에는 기본적인 데이터를 표현하기 위해
각 데이터를 표기하는 태그를 사용해야 한다.
텍스트를 표현하기 위해 사용하는 태그들은 어떤 것들이 있을까?

제목 태그

hn태그 - Heading

hn태그는 제목을 표기하는 태그이다
제목의 용도에 따라 1~6단계로 구분된다.

로고, 표지 제목

대제목, 단락제목

중제목, 페이지 제목

소제목, 문단 제목

문단 제목
주석 제목

본문태그, 문단태그

p 태그 - Paragraph

p태그는 문단이나 본문을 표기하는데 사용되는 태그이며
작성자가 여러 개의 문단으로 구성하고자 하면 여러 개의 p태그를 사용하여 표기한다

줄바꿈 태그

br 태그 - Break Line

br태그는 한 줄의 문장을 둘 이상으로 나눌 때 사용하는 태그이다.
여러 개의 br 태그를 사용하면 줄 사이에 여백이 생긴다.
따라서 여러 줄을 나누기 위해서 br태그를 반복적으로 사용하는 것은 지양

수평선 태그

hr 태그 - Horizontal Rule

hr태그는 한 줄의 수평선을 만들어 주는 태그이다.
속성값을 통해 선의 모양을 바꿀 수 있다.

텍스트를 한 줄로 표기하면서 강조하는 태그

strong 태그 - 굵게 강조할 텍스트 b 태그 - 굵게 강조할 텍스트
i 태그 - 이탤릭체로 표기하는 태그 em 태그 - 이탤릭체로 표기하는 태그
span 태그 - 줄 나눔없이 영역을 묶기
일부 글자만 스타일을 적용하고자 할 때 사용
u 태그 - 단지 밑줄 표시, 링크 표시하기 위한 용도 아님
s 태그 - 가격 표시할 때 사용하는 취소선 120,000원 100,000원
